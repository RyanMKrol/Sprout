# T222 — Square-tile home redesign + split Water / Full check-in

## 2026-06-02 — built (RyanMKrol / autonomous builder)

### What I did
- Redesigned `Sources/Views/HomeView.swift` from three stacked rectangular tiles into a
  **2-column `LazyVGrid` of square tiles** (`aspectRatio(1)`), on a grouped background so the
  cards read as distinct:
  - **My Plants** → pushes the list.
  - **Rooms** → pushes Rooms.
  - **Add plants** → presents the **T221 room-first add flow** (`AddFlowView` via `makeBasket`)
    directly from home, with the T208 "take photos?" follow-up replicated in `onDismiss`.
  - **Water your plants** → `startGuided(.due)` (plants due now).
  - **Full check-in** → `startGuided(.all)` (every plant).
  - Settings gear retained in the toolbar.
- Replaced the old single Water tile + `confirmationDialog` mode-chooser with the two distinct,
  signposted watering tiles, each launching the right `GuidedWateringCoordinator.Mode`.
- Factored the tile subtitle copy into a pure, testable `HomeTileText` enum.
- Added `Tests/HomeTileTextTests.swift` (4 cases) covering the My Plants / Water / Full-check-in
  subtitles.
- Docs in lockstep: README T222 row → ✅, TASKS.md box ticked, LIMITATIONS.md row for the
  duplicated photo-prompt flow (two sites for T223 to unify).
