# T009 — SproutSegmentedPicker

## Overview
The custom green-pill segmented control used by check-in, guided watering and room editing
(replaces `.pickerStyle(.segmented)` in redesigned screens).

## Do
1. Read `.harness/docs/designs/redesign-spec.md` §2.4 and the architecture contract.
2. Create `Sources/DesignSystem/SegmentedPicker.swift`: `SproutSegmentedPicker<T: Hashable>`
   with `init(selection: Binding<T>, options: [(value: T, label: String)])`. Track #E7E1D2,
   radius 15, inner padding 4; selected segment = brandGreen RoundedRectangle radius 11 with
   white `SproutFont.body(14.5,.semibold)` label and shadow 0 3 8 rgba(47,107,76,0.3), animated
   with `withAnimation(.spring(duration: 0.25))` on change; idle labels #6E7A63 medium. Equal
   widths (use a horizontal frame(maxWidth:.infinity) per segment).
3. Keep each option tappable with `.accessibilityIdentifier("segment.\(label)")`.

## Done when
- Project builds; `#Preview` shows a 3-option picker (Dry/Moist/Wet) with a selection bound to
  state and visibly moving pill.
