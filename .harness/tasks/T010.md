# T010 — SheetScaffold + SproutAlert

## Overview
Shared sheet chrome (grab handle + Cancel/Title/Save header on paper background, top radius 40)
and the frosted confirmation dialog (custom look — NOT SwiftUI .alert).

## Do
1. Read `.harness/docs/designs/redesign-spec.md` §2.8–2.9 and the architecture contract.
2. Create `Sources/DesignSystem/SheetScaffold.swift`: `SproutSheetHeader(cancelLabel: String? =
   "Cancel", title: String, confirmLabel: String?, confirmEnabled: Bool = true, onCancel:,
   onConfirm:)` — grab handle 40×5 rgba(60,66,58,0.2) centered above; Cancel left green
   body(17), title Bricolage `SproutFont.display(18)`, confirm right green body(17,.semibold),

   0.13 bg, icon full tint 20pt); title `display(19)`; message body(13.5) #6E7A63 centered; two
   equal buttons radius 13 — Cancel on rgba(60,66,58,0.08) ink label, confirm filled `tint`
   (destructive #C4553B) white label. Present via a `.sproutAlert(isPresented:) { config }`
   modifier that overlays at .infinity ignoring safe area with a fade+scale(0.95→1) transition.
