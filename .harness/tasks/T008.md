# T008 — Card styles: white card, bento tile, hero card, info banner, toolbar buttons

## Overview
The container vocabulary of the redesign: white cards, sage/oat bento tiles, the gradient
hero card shell, the soft-green info banner, and the circular toolbar button + green FAB.

## Do
1. Read `.harness/docs/designs/redesign-spec.md` §1.2 (sage/oat), §1.3, §2.5–2.7 and the
   architecture contract (Cards.swift).
2. Create `Sources/DesignSystem/Cards.swift`:
   - `.sproutCard()` modifier: white fill, radius 20 (parameterizable), hairline border
     rgba(34,39,31,0.05), card shadow.
   - `enum BentoStyle { case sage, oat }` + `.bentoTile(_ style:)`: fill/border/title/subtitle
     colors per spec §1.2, radius 24, padding 16, bento shadow.
   - `HeroCard<Content: View>`: radius-28 container with heroGradient fill, hero shadow,
     padding 22, and a `watermark: ChromeIcon` parameter rendered ~150pt white at 0.08 opacity,
     offset to bleed off a corner (parameter for topTrailing/bottomTrailing).
   - `InfoBanner(icon: ChromeIcon, text: String)`: rgba(47,107,76,0.10) fill, radius 16, icon +
     `SproutFont.body(13)` text in #2A6045.
   - `CircularToolbarButton(icon: ChromeIcon, tint: Color, action:)`: 42Ø white circle, shadow
     0 2 8 rgba(50,55,40,0.1), icon 16pt. `SproutFAB(action:)`: 40Ø brandGreen circle, white
     plus, shadow 0 4 12 rgba(47,107,76,0.3).

## Done when
- Project builds; a `#Preview` shows a white card, both bento tiles, a hero card with watermark,
  an info banner, a toolbar button and the FAB together on SproutTheme.paper.
- Names match the architecture contract; no screens changed yet.
