# T011 — RhythmBand (watering-rhythm visualization)

## Overview
The plant-detail "WATERING RHYTHM" band: a min→max track with a base-interval marker and a
"now" thumb — pure, testable position math plus the view.

## Do
1. Read `.harness/docs/designs/redesign-spec.md` §2.11 and the architecture contract.
2. Create `Sources/DesignSystem/RhythmBand.swift`:
   - `static func position(of value: Int, min: Int, max: Int) -> Double` — `(value-min)/(max-min)`
     clamped to 0…1; guard max<=min → 0.5.
   - View `init(minDays:maxDays:baseDays:effectiveDays:)`: eyebrow "WATERING RHYTHM"; "Every
     {effectiveDays} days" in `display(26)`; "MIN {n}d" / "MAX {n}d" labels body(11,.bold) taupe at
     the ends; 12pt-high track `LinearGradient(#E3EBD6→#B9D3A2)` radius 6; a 2pt ink@0.35 vertical
     marker at base position with "base {n}d" body(11) #9AA090 below; a 26Ø brandGreen thumb
     (3pt white border, white droplet ChromeIcon 11pt) at effective position with "now"
     body(11.5,.bold) green below. When |basePos−nowPos| < 0.08 show ONLY the thumb label.
   - Use GeometryReader for the track width; positions via the pure function.
3. Add `Tests/RhythmBandTests.swift`: position(7,4,14)==0.3; position(2,4,14)==0.0 (clamped);
   position(20,4,14)==1.0; position(5,5,5)==0.5; the label-collision predicate flips at 0.08.
