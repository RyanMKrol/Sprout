### What was done
Created `Sources/DesignSystem/SegmentedPicker.swift` with `SproutSegmentedPicker<T: Hashable>` generic component:

**Component features:**
- Binding-based selection state
- Generic over any Hashable type
- Track background: `#E7E1D2` (segmentedTrack) with radius 15
- Selected segment: brandGreen (`#2F6B4C`) RoundedRectangle with radius 11, inner padding 4
- Selected label: white text at 14.5pt semibold (SproutFont.body)
- Idle labels: `#6E7A63` (textMuted) at medium weight
- Shadow on selected pill: `0 3 8 rgba(47,107,76,0.3)`
- Animation: spring duration 0.25s on selection change
- Equal width segments via `frame(maxWidth:.infinity)` per option
- Accessibility identifiers: `segment.\(label)` for each option
- Includes a Preview (`SegmentedPickerPreview`) showing 3-option picker (Dry/Moist/Wet) with live selection binding
