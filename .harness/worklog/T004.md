## 2026-07-17

**Status: done**

### What was built

Created `Sources/DesignSystem/SproutIcon.swift` with:

1. **PlantIcon enum** (16 cases, all Phosphor glyphs):
   - flower, leaf, plant, pottedPlant, cactus, flowerTulip, flowerLotus, grains, clover, treePalm, tree, treeEvergreen, acorn, cherries, carrot, pepper
   - Each case maps to the correct rawValue (`ph-*`)
   - `var image: Image` returns template-rendered images
   - `static func default(forSpecies:)` maps plant species keywords to icons:
     - "cact"/"euphorb" → .cactus
     - "citrus"/"lemon"/"lime"/"cherry" → .cherries (moved before generic "tree" check to avoid matching "Lemon tree")
     - "palm" → .treePalm
     - "fern" → .leaf
     - "orchid"/"lotus" → .flowerLotus
     - "tulip" → .flowerTulip
     - "violet"/"cyclamen"/"begonia" → .flower
     - "pine"/"conifer" → .treeEvergreen
     - "ficus"/"tree" → .tree
     - "herb"/"basil"/"mint" → .grains
     - "carrot" → .carrot
     - "pepper"/"chili" → .pepper
     - "clover"/"oxalis" → .clover
     - "succulent"/"echeveria"/"crassula"/"aloe"/"haworthia" → .pottedPlant
     - else → .plant

2. **ChromeIcon enum** (33 cases, FontAwesome 6 solid icons):
   - seedling, gear, bell, bellSlash, droplet, plus, listCheck, chevronLeft, chevronRight, pencil, trash, house, couch, bed, bath, sun, arrowTrendUp, magnifyingGlass, circlePlus, shuffle, camera, circleCheck, check, arrowRight, lock, arrowsRotate, arrowUpFromBracket, copy, fileLines, circleInfo, xmark, utensils, mugSaucer
   - Each case maps to the correct rawValue (`fa-*`)
   - `var image: Image` returns template-rendered images

3. **Tests** in `Tests/SproutIconTests.swift`:
   - `testPlantIconImagesResolve()`: verifies all 16 PlantIcon rawValues resolve to UIImages
   - `testChromeIconImagesResolve()`: verifies all 33 ChromeIcon rawValues resolve to UIImages
   - `testPlantIconDefaultFor{Fern,Monstera,Cactus,Orchid,Succulent,Aloe,Tulip,TreePalm,Tree,Evergreen,Herb,Citrus,Carrot,Pepper,Clover,Flower}()`: tests the default(forSpecies:) mapping
   - `testPlantIconCodable()`: verifies PlantIcon can be encoded/decoded as JSON
